招聘岗位

.Net 开发工程师

工作职责：

1、参与公司网站产品/移动端产品/工具及平台产品的开发及维护

2、完成相应模块的需求、设计、编码、调试以及相关工作

3、负责相关技术文档编写

5、负责所开发项目的后期维护、Bug修复

6、持续学习，提升开发能力

任职资格：

1、3年及以上.NET平台开发经验，熟悉主流框架及应用，了解原理和机制

2、精通C#，熟悉 .NET Framework 框架及底层原理

3、熟练掌握ASP.NET MVC相关开发

4、熟悉MYSQL数据库，了解Redis、Memcached，对SQL优化有一定经验

5、熟悉.NET Core技术，有跨平台部署经验者优先

6、熟悉分布式技术原理，参与过服务化项目开发者优先

7、有项目整体架构经验者优先

高级测试工程师

工作职责：

1、参与项目的全流程质量把控，包括需求分析、设计评审、制定测试计划、设计和执行测试用例，并进行软件质量分析

2、能够发现项目过程中的痛点，并通过测试流程或测试工具等方法创新，努力提升项目质量和工作效率

3、能够灵活运用多种测试手段，解决测试工作中遇到的问题，并对他人进行指导

任职资格：

1、计算机或其它相关专业大专及以上学历，3年以上软件测试工作经验

2、精通软件测试理论、方法和过程，形成较好的测试思维体系

3、熟悉Java/Python/Shell等语言之一，掌握常见的自动化测试工具，熟悉主流自动化测试框架

4、具备较强的业务能力或技术钻研能力，具有良好的沟通能力和团队协作精神

5、熟练使用MySQL，可以编写常用sql脚本

6、有性能测试或工具开发经验者优先

7、有项目测试负责人经验者优先

高级web前端开发工程师

工作职责：

1、负责产品的设计研发工作

2、参与整体系统架构设计，找出系统瓶颈，改进系统算法，提高系统性能

3、参与项目的技术方案设计及技术评审

4、负责指导初中级工程师日常工作

任职资格：

1、计算机科学或相关专业毕业，3年以上前端工作经验

2、精通JavaScript、Css、Html，熟悉模块化开发，有框架设计能力

3、有使用过Vue、React等经验，有Web App/小程序开发经验，有线上项目（不少于3个）

4、有优化网站性能的实践及产出

5、具有一定的软件工程意识，对数据结构和算法设计有充分理解

6、具有良好的沟通能力和团队合作精神、优秀的分析问题和解决问题的能力，能承受较大工作压力

加分项：有Node开发经验，开源社区贡献过代码

Web前端专家

工作职责：

1、根据实际业务场景及需求，设计前端基础架构，并带领其他组员实现并持续优化

2、优化团队开发方式及流程、提升团队开发效率、推动团队技术发展

3、为前端团队引入创新的技术、创新的解决方案，用创新的思路解决问题

4、对页面性能优化、技术选型提供解决方案

5、为组员的技术成长制定目标、计划并帮助实施，提升整体技术水平

任职资格：

1、五年以上前端大型工程研发经验, 并有自己的技术沉淀

2、精通javascript、CSS、HTML，了解相关的前端生态

3、精通一类MV*的js framework，如AngularJS、React.js 或 Vue.js

4、对前端开发规范、工程化、组件化、测试有深入认识和实践

5、思路清晰，具备良好的沟通能力和学习能力，且具备强烈的进取心、团队合作精神

6、掌握Node.js，有后端开发经验的优先
